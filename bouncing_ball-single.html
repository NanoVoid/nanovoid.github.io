PROJECT SHOWCASE

Bouncing Ball

RigidBody (Godot 4.2.2)

A simple interactive Rigidbody in 2D. Left-click (green) adds difference in position between ball and mouse as velocity to Ball. Right-click (red) reduces Velocity.

Contains

  • RigidBody2D
  • Mouse Input
  • Velocity (Vector2)
  • Responsive Cursor based on Input
